# Staff Portal Overview

Welcome to the Zimbabwe e-Arrival Card staff documentation. This guide covers the features and functionality available to government personnel.

## User Roles

The system uses Role-Based Access Control (RBAC) with the following roles:

| Role | Description | Access Level |
|------|-------------|--------------|
| **Immigration Officer** | Border post officers | Review, approve, reject cards |
| **ZIMRA Officer** | Zimbabwe Revenue Authority | Customs declarations review |
| **Government Official** | Ministry personnel | Statistics and reports |
| **Admin** | System administrators | Full system access |
| **Supervisor/Manager** | Port supervisors | Officer performance metrics |

## Key Features

### For Immigration Officers

- **QR Code Scanning** - Quickly retrieve traveler information
- **Card Verification** - View complete arrival card details
- **Approval Workflow** - Approve or reject cards with notes
- **Search & Lookup** - Find cards by reference number

### For Supervisors & Managers

- **Officer Performance** - Track processing volumes by officer
- **Border Post Analytics** - Monitor activity at your post
- **Shift Reports** - View processing statistics by time period
- **Team Management** - Oversee officer workloads

### For ZIMRA Officers

- **Customs Review** - View all customs declarations
- **Flag for Inspection** - Mark cards for secondary review
- **Duty Calculation** - Process import duties
- **Currency Declarations** - Track declared amounts

### For Government Officials

- **Real-time Statistics** - Live arrival numbers
- **Trend Analysis** - Compare periods and identify patterns
- **Export Reports** - Download data in multiple formats
- **Nationality Breakdown** - See visitor demographics

## Security & Compliance

<Warning>
All system activity is logged for audit purposes. Unauthorized access or data misuse will be reported and may result in disciplinary action.
</Warning>

### Best Practices

- Never share your login credentials
- Log out when leaving your workstation
- Report suspicious activity immediately
- Follow data protection guidelines
